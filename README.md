# Projeto de Verificação e Validação de Software: Análise e Testes de Loja Virtual

Este repositório contém o relatório e a implementação de testes desenvolvidos para o **Trabalho 2 de Verificação e Validação de Software** da **Pontifícia Universidade Católica do Rio Grande do Sul (PUCRS)**, na Escola Politécnica. 

**Autores:** Augusto Knob e Bernardo Zenglein [cite: 4]  
**Ano:** 2026

---

## 📌 Sobre o Sistema

O sistema analisado é uma loja virtual composta por um catálogo de produtos, carrinhos e gerenciamento de pedidos. A aplicação é dividida em dois projetos independentes que se comunicam através de uma API REST:
* **Backend:** Desenvolvido em Java 11 com Spring Boot 2.2, Spring Security com JWT, Spring Data JPA/Hibernate e banco de dados PostgreSQL.
* **Frontend:** Um cliente construído em Angular 7 que consome a API backend.
